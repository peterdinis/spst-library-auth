## SPŠT Knižnica Auth Server

### Slovenská Verzia
#### Popis
Tento autentifikačný server slúži pre prihlásenie a registráciu používateľov do systému SPŠT Knižnica. Po registrácii môžu používatelia využívať nasledujúce funkcie:

- **Študenti:** Požičiavanie a vracanie kníh.
- **Učitelia a admini:** Spravovanie objednávok a administrácia systému.
